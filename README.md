# CANTEEN_MENU
This project is a Java console application that simulates a college canteen ordering and billing system. It demonstrates the use of nested switch–case statements, loops, and collections to build a structured menu, handle multiple orders, and generate a detailed bill.

Features
Displays a main menu with categories:

Tiffins (Veg + Non‑Veg options)

Lunch

Snacks (15 varieties)

Beverages

Each category opens a sub‑menu table with items and prices.

Customers can order multiple items in one run.

Stores all selected items in a list for billing.

Calculates subtotal, GST (5%), SGST (5%), and grand total.

Prints a receipt‑style bill with:

List of items ordered

Subtotal

GST & SGST

Final amount
